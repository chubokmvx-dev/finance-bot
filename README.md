# 💰 Фінансовий Telegram Бот

## Деплой на Railway (безкоштовно, без комп'ютера)

### Крок 1 — GitHub
1. Зайдіть на github.com → New repository
2. Назва: `finance-bot`
3. Завантажте всі файли

### Крок 2 — Railway
1. Зайдіть на railway.app
2. "New Project" → "Deploy from GitHub"
3. Оберіть `finance-bot`
4. У Variables додайте:
   - BOT_TOKEN = ваш токен від BotFather
   - ANTHROPIC_KEY = ваш ключ від Anthropic
5. Deploy!

### Крок 3 — Готово!
Напишіть боту /start і користуйтесь!
